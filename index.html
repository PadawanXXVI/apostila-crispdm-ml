Apostila CRISP-DM com Python para Machine Learning

Esta é uma apostila completa para quem deseja aprender Machine Learning com Python, seguindo a metodologia CRISP-DM.

📌 Objetivos

  • Entender todas as etapas da metodologia CRISP-DM
  • Aplicar ML com Python usando exemplos reais
  • Demonstrar boas práticas de modelagem, validação, otimização e interpretação

🧪 Projeto Didático com Titanic Dataset

A apostila inclui um projeto real com o famoso Titanic Dataset, usando pandas, scikit-learn e joblib.

🧭 Etapas da CRISP-DM

  1. Entendimento do Negócio
  2. Entendimento dos Dados
  3. Preparação dos Dados
  4. Modelagem
  5. Avaliação
  6. Implementação

📥 Como Usar

Abra este arquivo index.html no navegador para acessar a apostila offline.
